  #1.Project Overview

Sheira-M1W4 is a lightweight, three-page portfolio website that showcases talents, hobbies, and professional services. Built with Tailwind CSS, it offers a modern design with a responsive layout that works seamlessly across all devices.

  #2.Feature
    
 Mobile Menu - Hamburger menu toggle on all devices

 Service Cards - Clean grid layout with hover effects

 Contact Form - Functional contact form with validation

 Font Awesome Icons - Professional icon integration

 Tailwind CSS - Utility-first CSS framework

 No Dependencies - Minimal external dependencies

 Short & Clean Code - Optimized for readability
